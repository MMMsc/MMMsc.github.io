面向过程的程序设计

一个面向过程的程序包括以下两方面

  1. 对数据的描述。在程序中制定数据的类型和数据的形式，也就是数据结构
  2. 对操作的描述。操作步骤，也就是算法
    程序=算法+数据结构

C++输入和输出是用“流”的方式实现
用于输入和输出单个字符的函数，getchar()和putchar()

cout << setiosflags(ios::fixed) << setprecision(4); // 限定小数后4位
