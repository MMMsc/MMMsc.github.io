C++程序设计Day-0x05

引用

给一个变量起别名

  1. 引用没有定义(不是变量)，是一种关系型声明。部分配内存，占用的是同一片内存空间(a改变了b也随之改变,b变了a也变)
  2. 必须初始化，不可以作其他变量的引用
  3. 可以多次引用，一个变量有多个别名
  4. 主要是用做函数参数

常引用

const 数据类型 &引用名

const对象的引用必须是const类型的常量

const int &e 相当于 const int const e
普通引用 相当于 int const e
