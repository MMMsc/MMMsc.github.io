Mac 搭建Github+Hexo博客

安装前提

Node.js

Node.js

Git

Git

Hexo

$ sudo npm install -g hexo-cli

hexo-deployer-git

$ npm install --save hexo-deployer-git

开始搭建

初始化博客

新建一个空文件夹并cd进入
$ hexo init

生成网站

将网站生成为服务器可识别的标准HTML网站目录
$ hexo g

启动本地服务器

在本地服务器上启动网站 查看运行效果 默认地址 localhost:4000
$ hexo s

配置主题

可以在hexo的主题官网中自行选择Themes | Hexo进入到github的项目中，将项目clone到theme文件夹中，并命名为相应的主题名
本来想和大众不一样，自己看中了一款主题，但是该主题配置比较繁琐，还需要修改部分前端代码…于是，next真香，该主题为目前github上hexo主题板块最高star
