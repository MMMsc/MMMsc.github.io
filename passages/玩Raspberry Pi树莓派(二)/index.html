安装、配置Nginx

其中root和fastcgi_pass的地址一定要根据自己的路径来设定
配置完成后，重启nginx和php
$ curl localhost查看效果

修改端口

nginx和apache默认使用80端口，要进行修改

修改nginx

$ sudo vim /etc/nginx/sites-available/default
找到sever，将lisen后的80改成8080或其他
重启nginx
$ sudo service nginx restart

修改apache

$ sudo vim /etc/apache2/ports.conf
将listen 80改成8090或其他
重启apache
$ sudo service apache2 restart
