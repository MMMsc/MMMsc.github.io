C++程序设计Day-0x01

初步认识

一个c++程序可以由一个程序单位或多个程序单位构成；每一个程序单位作为一个文件；一个文件是一个编译单元。
一个程序单位包括：预处理命令、全局声明部分、函数

类体现数据的封装性和信息隐藏
信息隐藏是c++的一大特点
用带.h后缀的头文件时，不需要用using namespace std；声明

c++程序的编写步骤

  • 编写源程序，以.cpp为后缀
  • 对源程序编译，对应多个程序单位，生成多个目标程序，以.o(object)为后缀。编译的作用是对源程序逆行词法检查和语法检查
  • 连接目标文件，生成可执行的二进制文件，以.exe结尾
  • 运行程序
  • 分析运算结果

数据类型与表达式

小写字母转换为大写字母

字符串常量实际占用空间为原有+1，有’\0’作为结束符
符号常量 #define name price (要与常变量区分)

C++要求强制定义变量的目的

  • 程序中变量名使用正确
  • 变量确定类型，编译时分配相应的存储单元
  • 编译时，检测运算是否合法

赋值不可写成 float a=b=c=2;

常变量

也被称为只读变量
const int a=3; // 常变量不能被赋值
const int b=3+6, c=3*cos(1.5); //可以用表达式初始化
上式使用了cos()，头文件需要加上#include <cmath> 或 #include <math.h>

自加、自减只适用于变量

赋值过程的类型转换

强制类型转换 (类型名)(表达式)

  • 浮点型赋值给整型，舍弃小数
  • 整型赋值给浮点型，数值不变，以指数形式存放
  • double赋值给float，不能溢出
  • 字符型赋值给整型，将字符的ASCII码赋值
  • int、short、long赋值给char，只会将低8位送到char
  • signed赋值给unsigned，将存储单元连原有的符号位作为数值一起传送

赋值运算式 a = (b=5); //a和b都赋值为5

逗号运算符

表达式1，表达式2 //先求解表达式1，再求解表达式2
