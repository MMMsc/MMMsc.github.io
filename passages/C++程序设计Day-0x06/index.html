结构体

类型与变量是不同的概念，只能对结构体变量中的成员赋值，而不能对结构体类型赋值。编译时，只会为变量分配空间
成员也可以是结构体变量，如

结构体中的成员名可以和程序中的变量名相同，二者没有关系

结构体的初始化

结构体变量的引用

  1. 可以将一个结构体变量的值赋给另一个具有相同结构的结构体变量
    student1 = student2;
  2. 可以引用一个结构体变量中一个成员的值
    student1.num = 2333; // .优先级是**最高**的
  3. 必须逐级引用
    student1.birthday.month;
  4. 不能将一个结构体变量作为整体输入输出
    cout << student1;
  5. 可以对成员像普通变量一样运算
  6. 可以引用结构体变量成员的地址，也可以引用结构体变量的地址

指向结构体变量的指针

一个结构体变量的指针就是该变量占据的内存段的起始地址

  • stu.num
  • (*p).num
  • p->num
    三者等价

动态分配和撤销内存的运算符

C++用new和delete代替malloc和free，是运算符不是函数，所以效率高
new的一般格式：new 类型 [初值]
用new分配数组空间的时候不能指定初值，如果空间不足会返回NULL
delete的一般格式：delete[ ] 指针变量
