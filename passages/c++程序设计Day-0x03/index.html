C++程序设计Day-0x03

函数与预处理

引例及细节

  1. 该程序包括一个程序单元，作为源程序文件存放在计算机的外部存储器(磁盘)中。这个程序单元包括三个函数
  2. 用户定义的printstar和print_message函数没有函数参数，若给实参，报错
  3. 程序执行从main开始，由系统调用
  4. 所有函数是平行的
  5. 因为main在其他两个函数之后，所以不需要声明

函数从形式上分为有参函数和无参函数

形参和实参

  1. 定义函数时指定的形参，未函数调用时，不占用内存的存储单元
  2. 实参可以是常量、变量或表达式
  3. 定义函数时，必须定义参数类型
  4. 实参和形参类型相同或兼容
  5. 实参对形参是值传递，调用结束后，形参单元被释放，实参不改变

不允许对函数作嵌套定义，必须平行与独立

递归

直接或间接调用自身

局部变量和全局变量

局部变量：在函数内部定义，只在本函数范围内有效
主函数main中定义的变量也只在主函数内有效
形参也是局部变量
声明中的变量名也是无效的

全局变量：在函数外部定义的变量
如果在一个函数中改变了全局变量的值，会影响到其他函数
全局变量和局部变量同名，局部变量作为范围内，全部变量将会屏蔽
