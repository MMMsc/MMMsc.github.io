C++程序设计Day-0x08

类和对象进一步讨论

构造函数

对象的初始化

类的数据成员不能在声明类的时候初始化，类不是实体，是一种抽象类型

如果一个类中所有的成员都是公用的，则可以在定义对象的时候初始化

构造函数的说明

构造函数不需要用户调用，在建立对象的时候自动执行。名字必须和类名相同，不具有任何类型、没有返回值。

运行时，先建t1，并对其中成员赋初值0，再执行set_time；接着建立对象t2，赋初值0，但t2的成员不赋新的值，直接输出数据成员的初值

析构函数

名字是在类名前面加“～”。当对象的声明周期结束时，会自动执行析构函数。
析构函数的作用并不是删除对象，而是在撤销对象占用的内存前完成一些清理工作

this指针

当不同的对象的成员函数引用数据成员时，怎么确保引用的是指定对象的数据成员。在每一个成员函数中都包含一个特殊的指针，称为this，是指向本类对象的指针，值为当前被调用的成员函数所在对象的起始地址

静态成员

静态数据成员

在同类见的多个对象之前实现数据共享

height被所有对象共有，所有对象都可以引用。在内存中只占一份空间，如果值改变，那么所有对象中的值都改变。
若声明了类为定义对象，一般数据成员是不占有内存空间的。但是静态数据成员是在所有对象之外单独开辟空间，不随对象建立而分配，不随对象撤销而释放。
对静态数据成员的初始化
int Box :: height = 10;

静态成员函数

例如
static int volume();
如果要在类外调用
Box :: volume();或者a.volume();
静态成员函数与非静态成员函数的根本区别：非静态成员函数有this指针，静态成员函数没有
